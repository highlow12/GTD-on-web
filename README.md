# GTD-on-web

A web-based GTD (Getting Things Done) application built with React and Vite.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Tech Stack

- React 19
- Vite 7
- Modern JavaScript (ES Modules)
